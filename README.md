## Project Overview
This project is a smart digital system developed for *Modern Curtain* company to automate customer orders and inquiries through a professional website and a WhatsApp bot.

The system helps reduce manual work, organize customer requests, and improve the overall customer experience.

---

## Problem Statement
Modern Curtain receives a large number of customer messages daily through WhatsApp, which causes:
- Lost or forgotten requests.
- Delayed responses.
- Difficulty in organizing customer information.

---

## Solution
A complete automation system was built consisting of:

| Component | Description |
|----------|-------------|
| Website | Displays services, branches, Google Maps, and contact links |
| WhatsApp Bot | Automatically receives customer requests and replies |
| Database | Stores customer information securely |

---

## Website Features
- Modern and responsive design.
- Service showcase (Curtains, Smart Curtains, Custom Design).
- Google Maps for branch locations.
- Social media links (YouTube, TikTok, Instagram, WhatsApp).

---

## WhatsApp Bot Features
The bot collects the following information from customers:

| Field |
|-------|
| Customer Name |
| Curtain Type |
| Location |
| Activity Type (Residential / Commercial) |

The data is then saved automatically into the database.
